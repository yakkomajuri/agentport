# Getting Started

The easiest way to get started with AgentPort is to use our [Cloud version](https://app.agentport.sh) on the free plan (no credit card required).

If you'd like to self-host the open source version, check out our [docs on self-hosting](/self-host).

## Connecting to AgentPort

Instructions for connecting to common clients like Claude Code, Codex, OpenClaw, and more can be found on the "Connect" page in-app, but you can also refer to the specific pages for our [MCP](/connect/mcp) or [CLI](/connect/cli).

## Installing integrations

The easiest way to install integrations is via the AgentPort UI. The CLI and MCP also allow you to install integrations, so your agent can discover tools as it goes along, but approval policies can only be changed via the UI.

The CLI and MCP cannot change approval policies as that would allow agents to bypass them.

## Contributing an integration

You can contribute a new integration by submitting a PR to the [AgentPort repo](https://github.com/yakkomajuri/agentport).
